# Automation Scripts

A collection of automation scripts that I use for day-to-day tasks such as
file management, JSON processing, app configuration, and development utilities.

## Tech Stack
- Python
- JavaScript (Node.js)
- Shell scripts

## Folder Structure
- `python/` – Python automation scripts
- `javascript/` – Node.js based scripts
- `shell/` – Bash utilities
- `docs/` – Usage guides and documentation
